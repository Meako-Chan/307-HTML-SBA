My Work

Cloud Drive Sharing Manager

Collaborated in a team of four to create a cloud drive sharing manager that supported Google Drive and DropBox. Users were able to search through files intuitively and change permissions and sharing. Also implemented a snapshot system to store and retreive changes from a database.

Ranking Lists Website

Devloped a MERN stack website for creating, editing, and saving lists that were made to be published, viewed and commented on by other students. Created a login system to verify username & e-mail and used MongoDB API and Postman to request and store user information and lists in a database.

Keyboard Decoder

Took user input keyboard data to implement an "auto-correct" system. Using a keyboard-decoding algorithm to determine the likelihood of each word and compare success rates between algorithm and literal translation of keyboard inputs.
